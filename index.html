Tags

h1

El h1 tag forma parte de los 6 heading en HTML, h1 define al heading mas importante y de mayor tamaño.

Ejemplo h1:

<h1>Esto es un Header h1</h1>

Esto es un Header h1

p

El p tag define un parrafo.

Ejemplo p:

<p>Esto es un parrafo</p>

Esto es un parrafo

img

El img tag se usa para insertar una imagen en el HTML.

Ejemplo img:

<img src="https://cdo-curriculum.s3.amazonaws.com/media/uploads/img_tag.png" alt="img">

a

El a tag define un hipervínculo, que se utiliza para vincular de una página a otra.

Ejemplo a:

<a href="https://www.youtube.com/">Ir a Youtube</a> Ir a Youtube

Atributos

src

El atributo src especifica la ubicación (URL) de un recurso externo. src se puede utilizar en los siguientes tags:

  • audio
  • embed
  • iframe
  • img
  • input
  • script
  • source
  • track
  • video

Ejemplo src:

<script src="./index.js"></script>

style

El atributo style especifica estilos CSS.

Ejemplo style:

<p style="background-color: darksalmon; font-size: 20px;">ejemplo atributo style</p>

ejemplo atributo style

href

El atributo href especifica el link de resurso deseado (pagina web o archivo externo en caso del link tag). href se puede utilizar en los siguientes tags:

  • a
  • area
  • base
  • link

Ejemplo href:

<a href="https://www.youtube.com/">Ir a Youtube</a> Ir a Youtube
